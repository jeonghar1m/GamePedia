# 홍익대학교 졸업 작품

## 개요
영화DB API를 호출해 관련 정보를 시각화 해 사용자에게 정보를 제공해주는 웹사이트

## 사용 API
https://themoviedb.org (TMDB)

## 사용 프레임워크
* Front-End: React
* Back-End: Express
